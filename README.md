# 🎓 Teste Técnico – Desenvolvedor(a) Frontend (EdTech)

Página interativa desenvolvida como teste técnico, implementando fielmente o layout do Figma com HTML5, CSS3 e JavaScript puro (sem frameworks).

## 🔗 Demo

- Live: https://soulhiro.github.io/EdTech-Frontend/

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 (Custom Properties, Flexbox/Grid, desktop-first)
- JavaScript Vanilla (ES Modules)
- Swiper.js
- sessionStorage (persistência de estado)

## ✨ Funcionalidades

- 🎬 Player de vídeo (YouTube embed) responsivo
- 🖼 Imagem lateral responsiva
- 🎞 Slider de imagens com estados normal / hover / disabled
- 🔊 Player de áudio funcional
- ⭐ Recurso de destaque
- 🃏 Cards interativos com expansão/recolhimento
- 📝 Atividade discursiva com persistência completa (texto, feedback, estado dos botões)
- ✅ Atividade objetiva (checkbox) com persistência completa
- 📚 FAQ em accordion (elementos nativos `<details>`/`<summary>`)
- ♿ Acessibilidade: HTML semântico, labels, navegação por teclado, ARIA, contraste

## ▶️ Como Rodar

### 🌐 Acesse online (recomendado)

A versão publicada está disponível em:
**https://soulhiro.github.io/EdTech-Frontend/**

### 💻 Rodar localmente

1. Clone o repositório:
    ```bash
    git clone [url-do-repo]
    ```
2. Abra o arquivo `index.html` diretamente no navegador, ou use uma extensão como **Live Server** (VS Code) para servir localmente:
    ```
    Live Server → botão direito em index.html → "Open with Live Server"
    ```
    > Necessário servir via HTTP (não `file://`) por conta do uso de ES Modules (`type="module"`).
